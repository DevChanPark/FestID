1. 모바일 신분증 로그인에서 대시보드 진입

기본 흐름은 모바일 신분증 인증 성공 후 관리자 VC를 확인하고 대시보드로 이동

01 접속 관리자 웹에서 모바일 신분증 로그인을 시작합니다.
02 본인 인증 모바일 신분증 앱에서 DID 기반 신원 인증을 완료합니다.
03 관리자 VC 확인 발급된 관리자 권한 VC가 있으면 로그인 성공 처리합니다.
04 대시보드 선택된 축제의 관리자 대시보드로 진입합니다.
